# Семантика языка clvr
## Типы данных
 В нашем языке используется статическая типизация(фиксируется при объявлении).
 Бинарные операции возможны только между операндами одного типа.
 
- int: 32-битное целое число. Особенности:
    - Поддерживает арифметические операции: + - * / %
    - Поддерживает операции сравнения: == != < > <= >=
    - Деление / — целочисленное.
    - % возможен. 
    - Деление на 0 - ошибка времени выполнения
- float: вещественный тип данны. 64-битный IEEE-754 (double precision).Особенности:
    - ~15–16 значащих цифр точности
    - Поддерживает арифметические операции: + - * / %
    - Поддерживает операции сравнения: == != < > <= >=
    - Деление / — целочисленное.
    - % не возможен.
    - Деление на 0 - ошибка времени выполнения
- string: строковый тип данных. Особенности:
    - Заключается в двойные кавычки "".
    - Поддерживает конкатинацию через "+" и сравнение(==, !=).
    - Является последовательностью ASCII-символов.

## Условный оператор
- condition должна иметь тип bool.
- Если значение true — выполняется первая ветка.
- Иначе — ветка else (если есть).
- else прикрепляется к близжайшему if

## Цикл while
- condition должна быть bool.
- Проверяется перед каждой итерацией.
- Выполняется, пока условие истинно.
- Переменные внутри цикла объявлять нельзя.

## Ввод и вывод 

ввод(input):
- Для каждого идентификатора читается значение.
- Тип вводимого значения должен соответствовать типу переменной.
- Переменная должна быть объявлена.
- При неправильном вводе возникает ошибка времени выполнения

вывод(output):
- Вычисляет выражения слева направо.
- Выводит их представление.
- Разрешены все базовые типы и литералы.

## Ошибка ввода 

| Тип переменной | Неверный ввод                   |
|----------------|---------------------------------|
| int            | Дробное число, строка           |
| float          | строка                          |
| string         | отсутствует закрывающая кавычка |

## Переполнение
- int
    - Переполнение возникает при:
      сложении, вычитании, умножении, унарном отрицании,
      вычислении выражения, выходящего за диапазон
    - Переполнение int является ошибкой времени выполнения.
- float 
    - Если результат арифметической операции выходит за допустимый диапазон:
      результатом становится +∞ или -∞,
      выполнение программы продолжается
